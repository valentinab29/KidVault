# 💡 Concepto — KidVault
KidVault es una plataforma de seguros educativos que transforma un trámite financiero rígido en una experiencia humana. Está dirigida a padres de familia que desean asegurar el futuro profesional de sus hijos, pero temen no poder sostener el compromiso económico a largo plazo. El proyecto nació del insight de que el miedo a la incertidumbre frena el ahorro. Así, se replanteó la "bóveda" tradicional: en lugar de un objeto frío que encierra dinero, se rediseñó como un símbolo de progreso y continuidad que protege metas adaptándose a la realidad de cada familia. 

El flujo de la interfaz se estructuró a partir de un storyboard que mapea el viaje emocional del usuario, desde la desconfianza inicial hasta la tranquilidad al interactuar con la plataforma. Técnicamente, el reto de la aseguradora y la tienda virtual se resolvió con un diseño interactivo y responsivo usando CSS avanzado (Flexbox y Grid). La pasarela cuenta con pestañas dinámicas para métodos de pago (PSE, tarjetas), un formulario con validación de datos, carga de documentos y una previsualización de tarjeta en perspectiva 3D con reflejos en tiempo real. Esto humaniza el flujo, aportando claridad, transparencia y seguridad visual en cada aporte.

## ⚙️ Requisitos Mínimos para el Funcionamiento

Para que la aplicación web funcione de manera correcta y óptima, asegúrate de contar con las siguientes especificaciones:

* **🌐 Navegador Web Moderno (Preinstalado):** 
  * Google Chrome (v90+), Mozilla Firefox (v90+), Microsoft Edge o Safari. 
  * *Nota:* Se requiere soporte completo para CSS Flexbox, Grid Layout, propiedades de perspectiva 3D (`perspective`, `transform-style`) y pseudo-elementos.
* **💻 Servidor Local de Desarrollo (Recomendado):** 
  * Extensión *Live Server* en Visual Studio Code, XAMPP o NodeJS. Esto evita restricciones de políticas de origen (CORS) al procesar o cargar ciertos recursos locales.
* **📱 Resolución de Pantalla:** 
  * Optimizado para pantallas de escritorio (1024px en adelante).
  * Cuenta con diseño adaptativo (*Responsive Design*) mediante Media Queries para una visualización correcta en dispositivos móviles (hasta 320px).
